Pipeline

One reusable loop for real-world RL

01

HIL Collection

Collect trajectories with explicit intervention and outcome labels.

02

Dataset Report

Audit quality before training, not after failure.

03

Value Training

Learn value signals from real outcomes and long-horizon context.

04

Advantage-Indicator Annotation

Write value/advantage/indicator back into the dataset.

05

Policy Iteration

Train indicator-conditioned policy and return to real-world rollouts.

Repro Entry

Minimal commands to run the released loop

# 1) HIL recording
export DATASET_REPO_ID=<your_dataset_repo_id>

lerobot-human-inloop-record \
  --robot.type=so100_follower \
  --teleop.type=so100_leader \
  --dataset.repo_id=${DATASET_REPO_ID}

# 2) Dataset report
lerobot-dataset-report --dataset ${DATASET_REPO_ID} --json

# 3) Value train
lerobot-value-train \
  --dataset.repo_id=${DATASET_REPO_ID} \
  --value.type=pistar06 \
  --steps=2000

# 4) Value infer + indicator labels
lerobot-value-infer \
  --dataset.repo_id=${DATASET_REPO_ID} \
  --inference.checkpoint_path=outputs/value_train/evo_value_demo \
  --acp.enable=true

# 5) Indicator-conditioned policy train
lerobot-train \
  --dataset.repo_id=${DATASET_REPO_ID} \
  --acp.enable=true
